Gran Turismo (1) Car Code Chart

I took quite a few days recently to put together this chart of car codes for GT1. This chart also represents, as far as I can tell, an absolutely complete car list for the game.

Extrapolated from some internal tables in the game, this chart is in order by the File field--the internal filename code used instead of the garage code by many parts of the game. For example, internal prize lists use those codes, and the car appearing in a special model showroom is selected by one of those codes.

This chart gives complete parameters which can be used with some of my former hints (from over three years ago) to do things like populate used car lots, alter prize lists, and change the cars in special model showrooms.

I created these codes by examining GT1 version 1.1, but I would suspect that the identical list was used in version 1.0. (Eventually I will try to confirm that, now that I have a version 1.0 disk). I also took care to produce a name list which used the North American names of the cars, rather than the mostly Japanese names distributed at many Game Shark code sites. I give the actual in-race names used in the North American game, and you will see that I thereby indicate precisely how those race-modified Subara names are mixed up--a minor bug you may have noticed.

In testing, I tried driving the three hidden cars which I indicate lock the game up. (An extra Celica GT-Four which I have never read about anywhere else, and the comparatively well-known two hidden del Sols). You can actually race any of these cars if it happens to be your current car, but you cannot view it in your garage without locking the game up. This means the only way to select one of them is to use a Game Shark. That is, if you received one of these cars on a save from someone else, the only way you could race it without a Game Shark would be if it already happened to be the current car for that save.
