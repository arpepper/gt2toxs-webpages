Doing Weird Stuff with My Game Shark

Well, as I said, I bought a GameShark. I have discovered a few neat things, building on Code Master's Game Shark Code Creators's Club, specifically Gran Turismo Version 1.1 Codes.

There's something interesting, isn't it? Apparently there were two versions of Gran Turismo released in North America, distinguishable only by their actual memory use. I bought my first copy in December 1998, and it was already version 1.1, so I think the second version must have come out fairly soon after the initial release, and I think version 1.0 must actually be relatively rare.

If you're not from North America, I would say the place to look for Xplorer type codes is www.psxdomain.com, specifically:

  • the Pal Gran Turismo cheat codes
  • the Japanese Gran Turismo cheat codes
  • the US Gran Turismo V1.0 cheat codes

I'll probably add some details to this entry, or else in other entries.
