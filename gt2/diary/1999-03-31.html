One can visually scan this chart, starting from the ratio one has, looking for close values in either direction. In addition, one should look for an identical or close ratio to the starting value, and search around there.

A good enhancement would be to show the chart in shades of a colour, with darkness of the shade corresponding to the size of the number.

Someone good at using a slide rule should be able to accomplish much this same effect with a slide rule. I thought I'd seen a page on the web explaining that, but Alta Vista finds no matches for +gran +turismo +"slide rule" (nor if I say "sliderule", nor "slide-rule"). Looking for +gran +turismo +slide +rule finds 17 pages, and the ones I've looked at are using "rule" and "slide" separately.

Of course, someone good at using a slide rule might be really good with ratios, too, and be able to know that the next highest ratio after 10/7, using the integers 1..15 is 13/9...

Adrian.

#!/xhbin/perl5 -w

printf("%44s\n","Rear");
printf(" F ");
for ($r = 1; $r <= 15; ++$r) {
        printf("%5d", $r );
}
printf("\n");
for ( $f = 1; $f <= 15; ++$f) {
        printf("%2d: ",$f);
        for ($r = 1; $r <= 15; ++$r) {
                printf("%5.1f", $f * 100.0  / ($f + $r));
        }
        printf("\n");
}
