(GT1 Version 1.1) Lightweight Series Eligibility List, Primarily for GameShark Use

When I previously documented the parameters for the Lightweight Series, I did not give the list of elgible cars.

This list is accessible in memory during "shopping mode" at 80053AFC. The addresses shown, rather than being addresses of pointers to the car code strings are the actual addresses of the strings themselves. The strings are all aligned on four-byte boundaries (i.e. padded with zeroes to eight bytes each):

Note that any car which can be entered in normal form can be entered in race-mod form, too. Note that, apart from the Demio Aspec, no other prize cars can be entered in this series, not even the prizes you win from it! A few of the cars in the above list are not explicitly mentioned in the official Gran Turismo Reference Manual from Polyphony.

At 80058310 there is a table of pointers referring to these strings. It begins with the count 38(hex) or 56, followed by the above pointers, in reverse order. Strange fact: the above list has 68 entries, so perhaps the count does not refer to it.

If you use a GameShark Pro to alter one of the above strings, that car will no longer be eligible. If you in fact set the string to match some other car, that car can then enter the series.
