Ducati World Racing Challenge Complete Track List

By various analysis, I determined that the following is the list of all tracks available in Ducati World Racing Challenge.

I have included the zero-origin track number in hex as well as decimal. (The hex number can be used in conjunction with some GameShark codes and techniques).

Using some codes, you can arrange various types of races on all these tracks. However, the license test tracks, while the game won't crash if you use them, result in most cases in a race which cannot be finished.
