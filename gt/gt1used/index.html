This Used Car Finder will search and display information for all used cars available. There are 60 line-ups, one for each 10 day period from day 0 to day 590. After day 600, the cycle starts over again.

Many thanks to the many people who inspired this page, most noticeably Nick Albright. The data is from the North American version of the game.

This page is sort of a tutorial. Clicking one of the following items will execute a Used Car Finder query as if you had filled out the form.

The filled-in form will be shown on the generated page, followed by the results of the query. Study the form to learn how the query is done. You can modify it to try a slightly different query. Note that both form and results look much better if the window is reasonably wide.

Or, if you already know what you're doing, go straight to the Used Car Finder to start a brand-new query.

  • Show all used Nissans available for days 50 to 59.
  • Find those hard-to-find used Nismos in the available data.
  • Give a count of all used Silvias available on each day in the available data.
  • Give a count of each model of used Silvia available on each day in the available data.
  • Give as the count the total number of distinct records in the available data.
  • Give the count of used cars for each Manufacturer.
  • Give the total number of starter cars for each Manufacturer.
  • Show all the starter cars available at the beginning of the game, sorted by name.
  • Sort those same cars by price instead.
  • Find the twenty cheapest cars in the game.
  • Exclude Silvias from that search.
  • Limit that search to day 20 or lower.
  • Find the twenty cheapest Subarus.
  • Find the twenty cheapest Imprezas.
  • Find the twenty cheapest Imprezas, excluding wagons.
  • Find the twenty most expensive used cars.

Sorry, this tutorial is not great, but it was easy to do. I also found it very useful when doing my own testing.

When you see the form, basically fill in at the top of the form the characteristics, and the value you want them to match how. The tricky part is selecting the fields you want to see in the result. This form makes use of "multiple selection" for those selectors, and how to do this varies depending upon the version of the browser you are using. The Netscape we have here on UNIX interprets any mouse selection in those lists as a "toggle", while Netscape on my PC requires that the "control" key be pressed at the same time to work like that. IE on my iMac requires that the "apple" key be pressed. The "sort" menus work in a similar fashion, but can be ignored for your first queries; output will be sorted by the selected fields.
